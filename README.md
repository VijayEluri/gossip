Building
--------

### Requirements

* [Maven](http://maven.apache.org) 2.x
* [Java](http://java.sun.com/) 5
* [JLine2](http://github.com/jdillon/jline2)

Check out and build:

    git clone git://github.com/jdillon/gossip.git
    cd gossip
    mvn install
